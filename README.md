# Netflix Data Analysis Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset.

## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn

## Analysis Performed
Data Cleaning
Exploratory Data Analysis
Trend Analysis
Visualization

## Key Insights
- Netflix has more Movies than TV Shows
- Content production increased significantly after 2016
- USA produces the most Netflix content
